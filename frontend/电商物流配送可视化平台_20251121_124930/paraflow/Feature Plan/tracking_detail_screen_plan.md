# 物流追踪页
用户查看包裹的实时位置、预计送达时间和完整的物流历史记录,通过地图动画和时间轴展示配送进度。

Layout Hierarchy:
- Header(全宽):顶部导航栏
- Content Container(位于Header下方):主内容区域(分为上下两栏)

## 顶部导航栏
- 平台Logo
- 返回查询页按钮
- 刷新按钮
- 用户头像
- 退出登录按钮

## 主内容区域 - 上栏(地图追踪)
- 订单信息概览卡片:
  - 订单号
  - 当前物流状态标签
  - 预计送达时间(大字体突出)
  - 快递公司名称
  - 快递单号
- 交互式地图容器:
  - 起点标记(发货地址)
  - 终点标记(收货地址)
  - 配送路线(虚线)
  - 包裹当前位置标记(车辆图标,带动画效果)
  - 地图缩放控制
  - 地图图层切换(卫星/道路)
- 实时状态提示:
  - 包裹正在配送中提示文字
  - 距离收货地址距离显示
  - 最后更新时间显示

## 主内容区域 - 下栏(物流时间轴)
- 模块标题(物流详情)
- 物流时间轴组件:
  - 已签收节点(如已完成):
    - 时间戳
    - 状态描述
    - 签收人信息
    - 完成图标
  - 派送中节点(如当前状态):
    - 时间戳
    - 状态描述
    - 配送员信息
    - 进行中图标(高亮)
  - 运输中节点:
    - 时间戳
    - 状态描述
    - 运输站点信息
    - 完成图标
  - 已揽收节点:
    - 时间戳
    - 状态描述
    - 揽收站点信息
    - 完成图标
  - 订单创建节点:
    - 时间戳
    - 状态描述
    - 完成图标
- 收货地址信息卡片:
  - 收货人姓名
  - 联系电话
  - 详细地址
- 操作按钮组:
  - 联系配送员按钮
  - 修改收货地址按钮
  - 投诉/反馈按钮
