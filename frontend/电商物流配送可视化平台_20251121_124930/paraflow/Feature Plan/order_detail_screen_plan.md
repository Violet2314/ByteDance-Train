# 订单详情页
展示单个订单的完整信息,包括收货人、商品详情、当前物流状态,支持执行发货操作。

Layout Hierarchy:
- Header(全宽):顶部导航栏
- Content Container(位于Header下方):
  - 左侧边栏
  - 主内容区域

## 顶部导航栏
- 平台Logo
- 导航菜单(订单管理、配送管理、数据看板)
- 全局搜索框
- 通知图标
- 用户头像

## 左侧边栏
- 首页导航
- 订单管理(当前选中)
- 配送管理导航
- 数据看板导航
- 设置选项

## 主内容区域
- 页面标题与面包屑导航(订单管理 > 订单详情)
- 返回订单列表按钮
- 订单基本信息卡片:
  - 订单号
  - 订单状态(状态标签)
  - 创建时间
  - 订单金额
  - 发货按钮(待发货状态下可用)
- 收货人信息卡片:
  - 收货人姓名
  - 联系电话
  - 收货地址
  - 备注信息
- 商品详情卡片:
  - 商品列表(商品缩略图、名称、规格、数量、单价、小计)
  - 商品总数量
  - 商品总金额
- 物流状态卡片:
  - 当前物流状态
  - 快递公司信息
  - 物流单号
  - 物流时间轴(已揽收、运输中、派送中、已签收)
  - 查看完整物流追踪链接
