# 订单管理仪表盘
商家查看、筛选和管理所有订单,支持按状态筛选、排序和批量操作,快速完成发货任务。

Layout Hierarchy:
- Header(全宽):顶部导航栏
- Content Container(位于Header下方):
  - 左侧边栏
  - 主内容区域

## 顶部导航栏
- 平台Logo
- 导航菜单(订单管理、配送管理、数据看板)
- 全局搜索框
- 通知图标
- 用户头像

## 左侧边栏
- 首页导航
- 订单管理(当前选中)
- 配送管理导航
- 数据看板导航
- 设置选项

## 主内容区域
- 页面标题与面包屑导航
- 订单统计卡片组:
  - 待发货订单数量
  - 运输中订单数量
  - 已完成订单数量
  - 异常订单数量
- 订单筛选与操作栏:
  - 状态筛选器(全部/待发货/运输中/已完成)
  - 时间范围选择器
  - 搜索框(订单号/收货人)
  - 排序选项(按创建时间/金额)
  - 批量操作按钮(批量发货/导出)
- 订单列表表格:
  - 复选框列
  - 订单号列
  - 收货人信息列
  - 商品信息列(缩略图、名称、数量)
  - 订单金额列
  - 订单状态列(状态标签)
  - 创建时间列
  - 操作列(查看详情/发货按钮)
- 分页控件
