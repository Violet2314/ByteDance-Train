# 物流查询页
用户通过订单号查询包裹的物流信息,提供简洁的查询入口和历史记录功能。

Layout Hierarchy:
- Header(全宽):顶部导航栏
- Content Area(位于Header下方):主内容区域(全屏居中)

## 顶部导航栏
- 平台Logo
- 返回首页链接
- 用户头像
- 退出登录按钮

## 主内容区域
- 页面标题与说明文案
- 查询表单区域:
  - 订单号输入框(大尺寸)
  - 查询按钮(醒目样式)
  - 输入提示文字(如何找到订单号)
- 快速查询功能:
  - 最近查询记录列表(最多5条):
    - 订单号
    - 查询时间
    - 当前物流状态标签
    - 快速查看按钮
  - 清空历史记录按钮
- 帮助信息区域:
  - 常见问题链接
  - 联系客服按钮
  - 使用说明文字
