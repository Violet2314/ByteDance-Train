# 数据看板页
商家通过可视化图表分析物流数据,包括区域订单密度热力图、配送时效统计、异常订单监控,支持运营决策优化。

Layout Hierarchy:
- Header(全宽):顶部导航栏
- Content Container(位于Header下方):
  - 左侧边栏
  - 主内容区域

## 顶部导航栏
- 平台Logo
- 导航菜单(订单管理、配送管理、数据看板)
- 全局搜索框
- 通知图标
- 用户头像

## 左侧边栏
- 首页导航
- 订单管理导航
- 配送管理导航
- 数据看板(当前选中)
- 设置选项

## 主内容区域
- 页面标题与数据更新时间
- 时间范围选择器(今日/本周/本月/自定义)
- 刷新按钮
- 核心指标卡片组:
  - 总订单量(数值+环比增长率)
  - 平均配送时长(数值+环比变化)
  - 配送完成率(百分比+趋势图标)
  - 异常订单数量(数值+环比变化)
- 区域订单密度热力图模块:
  - 模块标题
  - 地图容器(热力图层叠加)
  - 订单密度图例(低/中/高)
  - 区域筛选器(按省/市筛选)
- 配送时效分析模块:
  - 模块标题
  - 配送时长分布条形图:
    - X轴:配送时长区间(0-12h, 12-24h, 1-2天, 2-3天, 3天以上)
    - Y轴:订单数量
  - 快递公司对比折线图:
    - X轴:日期
    - Y轴:平均配送时长
    - 多条折线(不同快递公司)
  - 数据导出按钮
- 异常订单监控模块:
  - 模块标题
  - 异常订单实时列表:
    - 订单号列
    - 异常类型标签(超时/滞留/投诉)
    - 收货地址列
    - 异常时长列
    - 当前状态列
    - 快速操作按钮(查看详情/联系客户)
  - 异常类型分布饼图:
    - 配送超时比例
    - 包裹滞留比例
    - 客户投诉比例
  - 异常趋势折线图:
    - X轴:日期
    - Y轴:异常订单数量
